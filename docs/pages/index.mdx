# Twapi

Twapi is a suite of packages that interact with twitch's backend services using **node.js** and **typescript**.

## Guides

| Package                         | Description                   |
| :------------------------------ | :---------------------------- |
| [EventSub](eventsub/quickstart) | Interact with twitch eventsub |
| [Helix Api](helix/quickstart)   | Call the twitch helix api     |
| [PubSub](pubsub/quickstart)     | Interact with twitch pubsub   |
| [Chat](chat/quickstart)         | Interact with twitch irc chat |

<Callout>
  We tried to leverage typescript the most so it is **recommended** to use
  typescript with any of the above libraries.
</Callout>
